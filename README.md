#  TransCesar

Sistema de gestión de transporte  enfocado en la administración de vehículos, pasajeros, rutas y tickets.

---

## 📌 ¿Qué hace este sistema?

TransCesar permite:

* Registrar y gestionar **vehículos** (bus, buseta, microbús)
* Administrar **pasajeros y conductores**
* Gestionar **rutas** con información detallada
* Generar y calcular **tickets de viaje**
* Aplicar **descuentos automáticos** (adulto mayor)
* Ajustar tarifas en **días festivos**

---

## 🧠 ¿Cómo funciona?

El sistema está construido bajo una arquitectura por capas:

* **Modelo** → Estructura de datos (Vehículos, Personas, Tickets, Rutas)
* **DAO** → Acceso y persistencia de datos
* **Service** → Lógica de negocio
* **View** → Interfaz en consola


---

## 🚀 Funcionalidades clave

* ✔️ Asignación de rutas estructuradas a vehículos
* ✔️ Cálculo automático de edad y descuentos (60+ años → 30%)
* ✔️ Incremento del 20% en tarifas en días festivos
* ✔️ Separación clara de responsabilidades (arquitectura limpia)
